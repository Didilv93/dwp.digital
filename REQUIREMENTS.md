Instructions for the Front-End Developer (Offline Test):
Objective
• Build a webpage with a form, which asks for Name, E-mail, Date of birth and Phone number.
• When the user submits the form, validate and display the data below the form on the same page.
• Each new submission should be appended to the existing data, and it should be possible to remove it.
• You are free to style the page however you like.
 
Code Test Rules
1. Write your own HTML.
 
Please do not use govuk-frontend or any other Frontend framework.
1. Write your own CSS.
 
You may use tools to generate your CSS eg. Sass or PostCSS
You should not use a CSS framework eg. Tailwind or Bootstrap.
1. Write your own vanilla JavaScript (ES6+) or TypeScript.
 
You may use build tools to transpile and/or run your code.
You should not use a framework eg. React or Angular.
1. Write your own validation.
 
You should not rely on the browser's built-in form validation.
1. Write test coverage.
 
You may use a testing framework.
1. Make sure it is accessible to WCAG 2.2 AA
2. Make sure it works in the browsers listed on this page: https://www.gov.uk/service-manual/technology/designing-for-different-browsers-and-devices
